Bookmarklet Install Instructions

Here is how to install the bookmarklet -

  1. Right click this link and copy the link address
  2. Bookmark this page, then click "edit bookmark"
  3. Paste the link address of the above link into the URL part of the bookmark and click save
  4. Whenever you click that bookmark, it should inject Hot Corn into your current page. Does not work with Google Classroom.
